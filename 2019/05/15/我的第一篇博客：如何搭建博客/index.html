我的第一篇博客：如何搭建博客

https://github.com/liuwen766/liuwen766.github.io.git

这是一个Hexo博客，技术成熟，遇到问题可以网上找解决方案。

安装Node.js
1.Node.js下载（Node.js+nmp包）
2.查看版本 node -v nmp -v
3.下载安装淘宝镜像 nmp install -g cnmp –registry-http：//registry.npm.taobao.org
4 cnpm
安装hexo
5.cnpm install -g hexo-cli 安装完成
6.hexo -v
使用hexo创建bolg
7.mkdis blog pwd
8.sudo hexo init
9.ll查看hexo框架
10.hexo s 启动
11.根据默认端口访问博客
新建博客文章
12.hexo n “我的第一篇博客：如何搭建博客”
13.cd source/_posts/
14.vim 我的第一篇博客：如何搭建博客
内容
15.pwd （在blog文件夹里）
16.hexo clean hexo g
17.hexo s 博客写好啦
远端部署：github
18.登录github 建立新仓库：liuwen766.github.io
git部署插件
19.pwd （在blog文件夹下）
20.cnpm install –save hexo-deployer-git（cnpm install hexo-deployer-git –save）
warning忽略
21.设置config文件 ll（查找改文件）
22.vim _confid.yml (type查看/输入文件名直接打开编辑)
deploy:
type: git
repo: https://github.com/liuwen766/liuwen766.github.io.git
branch: master
23.部署到远端 hexo d
部署后 输入github的账号：liuwen766 和密码：
24.完成 可以输入：liuwen766.github.io 来查看

更换主题部署
25.github.com/litten/hexo-theme-yilia
26.在blog 文件夹下 先下载，输入
git clone https://github.com/litten/hexo-theme-yilia themes/yilia
github.com/litten/hexo-theme-yilia
27 查看
cd themes/
ll 里面有yilia
28.修改blog文件夹下的config
在倒数第四行的 theme：yilia（有空格）
29.hexo clean hexo g hexo s （本地查看） localhost:4000
30.hexo d 推到远端github中 （github网址查看） liuwen766.github.io
